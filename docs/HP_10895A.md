# HP 10895A Laser Axis Interferometer

Author: Tim Mooney (1995)

The Hewlett-Packard 10895A is a VMEbus laser axis board for laser
interferometer measurements. This device support provides access to the
board's status, position, control, and command registers through bi, bo, mbbi,
mbbo, longin, and longout records.

## Platform Support

vxWorks only.

## Source Files

| File | Description |
|------|-------------|
| `vmeApp/src/devXxHP10895LaserAxis.c` | Device support for all record types |

## Configuration

Call `devHPLaserAxisConfig` in the IOC startup script before `iocInit`:

```
devHPLaserAxisConfig(ncards, a16base)
```

| Argument | Description | Default |
|----------|-------------|---------|
| `ncards` | Number of cards | 0 |
| `a16base` | A16 base address | `0x1000` |

**Example:**

```
devHPLaserAxisConfig(1, 0x1000)
```

## Supported Record Types

All record types use DTYP `"HP interferometer"` with VME_IO link type.

### bi / bo Records

| Type | Signal | Register |
|------|--------|----------|
| bi | 0-15 | Status register (16-bit) |
| bo | 0-15 | Control register |
| bo | 16-31 | Command register |

### mbbi / mbbo Records

| Type | Signal | Register |
|------|--------|----------|
| mbbi | 0-15 | Status register |
| mbbo | 0-15 | Control register |
| mbbo | 16-31 | Command register |

### longin Records

| Signal | Register |
|--------|----------|
| 0 | Status |
| 1 | Position |
| 2 | Sampled Position |

### longout Records

| Signal | Register |
|--------|----------|
| 0 | Control |
| 1 | Command |
| 2 | Offset |
| 3 | Clip Mask |
| 4 | Window Mask |

## Database Files

| File | Description |
|------|-------------|
| `vmeApp/Db/HPLaserAxis.db` | Full set of records for one laser axis card |

### Database Macros

| Macro | Description |
|-------|-------------|
| `$(P)` | PV prefix |
| `$(Q)` | Instance prefix |
| `$(C)` | Card number |

### Key PV Names

| PV Suffix | Type | Description |
|-----------|------|-------------|
| `RefLOL` | bi | Reference loss-of-light |
| `MeasLOL` | bi | Measurement loss-of-light |
| `Overflow` | bi | Counter overflow |
| `RefPres` | bi | Reference signal present |
| `MeasPres` | bi | Measurement signal present |
| `UserLED` | bo | User LED control |
| `Sample` | bo | Sample position command |
| `Reset` | bo | Reset command |
| `SPos` | longin | Sampled position |
| `Pos` | longin | Current position |
| `Offset` | longout | Position offset |

## iocsh Script

```
< $(VME)/iocsh/HP_10895A.iocsh
```

### Script Macros

| Macro | Required | Default | Description |
|-------|----------|---------|-------------|
| `PREFIX` | Yes | -- | IOC prefix |
| `INSTANCE` | Yes | -- | Instance prefix |
| `VME` | Yes | -- | Location of vme module |
| `ADDRESS` | First call | -- | A16 base address |
| `MAX_CARDS` | No | 1 | Total number of cards |
| `CARD` | No | 0 | Card number |

## Operator Displays

| File | Format |
|------|--------|
| `vmeApp/op/adl/HPLaserAxis.adl` | MEDM |

Autoconverted displays are available in the `bob/`, `edl/`, `opi/`, and `ui/`
subdirectories.

## Debug Variable

```
var devHPLaserAxisDebug 0
```

| Level | Output |
|-------|--------|
| >= 5 | Hardware initialization / probe messages |
| >= 10 | Record initialization errors |
